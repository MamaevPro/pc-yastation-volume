# Управление громкостью Яндекс Станции с ПК через HomeAssistant

Автоматически изменяет громкость колонки Яндекс Станция в зависимости от воспроизведения звука на компьютере.

## Возможности
- 🎵 90% громкости при воспроизведении звука
- 🔇 60% громкости при остановке звука
- 🎮 Работает с играми, музыкой, видео
- 📡 Управление Bluetooth через Home Assistant

## Требования
- Windows 10/11
- Колонка Яндекс Станция
- Home Assistant

## Настройка
Измените CONFIG в pc-yastation-volume.py:
- Адрес и токен Home Assistant
- Entity ID Яндекс Станции
- Уровни громкости

## Сборка и запуск
```
pip install -r requirements.txt
build.cmd
dist/pc-yastation-volume.exe
```
